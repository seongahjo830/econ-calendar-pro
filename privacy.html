시행일: 2026년 6월 4일 · v1.0
한 줄 요약 — 본 앱(경제캘린더)은 사용자의 어떤 개인정보도 수집·저장·전송하지 않습니다. 모든 데이터는 앱 안에 하드코딩되어 있으며, 오프라인에서도 완전히 동작합니다.

1. 수집하는 정보

없음. 본 앱은 다음을 포함한 어떠한 사용자 정보도 수집하지 않습니다:

  • 이름·이메일·전화번호 등 식별정보
  • 위치 정보 (GPS·IP 기반 모두)
  • 기기 식별자 (광고 ID·기기 ID)
  • 사용 패턴·앱 사용 로그
  • 연락처·캘린더·사진 등 기기 내 데이터
  • 마이크·카메라·센서 데이터

2. 외부 서비스

본 앱은 다음 외부 서비스를 사용하지 않습니다:

  • Google Analytics · Firebase Analytics 등 분석 도구
  • 광고 네트워크 (AdMob 등)
  • SDK를 통한 제3자 데이터 공유
  • 외부 API 호출 (모든 데이터는 앱에 내장)

3. 권한

본 앱이 요구하는 안드로이드 권한은 다음과 같습니다:

  • 인터넷 권한 — 앱 업데이트 확인 목적으로만 사용. 사용자 데이터 전송 없음.

4. 어린이 개인정보 보호

본 앱은 만 13세 미만 어린이를 포함한 모든 연령에 안전합니다. 어린이로부터 개인정보를 수집하지 않습니다.

5. 데이터 보관·삭제

수집하는 데이터가 없으므로 보관·삭제 정책이 해당하지 않습니다. 앱을 삭제하면 모든 로컬 캐시(서비스 워커 캐시)가 함께 제거됩니다.

6. 정보 보안

본 앱은 사용자 정보를 처리하지 않으므로 정보 유출 위험이 원천적으로 없습니다. 앱 자체는 HTTPS를 통해서만 배포·업데이트됩니다.

7. 정책 변경

본 정책이 변경될 경우 앱 내 설정 화면과 본 URL에 게시합니다. 중요한 변경(데이터 수집 시작 등)이 있다면 앱 업데이트 안내와 함께 명확히 고지합니다.

8. 면책 조항

본 앱이 제공하는 경제지표·일정·시나리오·팁은 정보 제공 목적이며, 투자 권유나 자문이 아닙니다. 발표 일정은 각 기관의 공식 발표에 따라 변동될 수 있으며, 실제 발표와 차이가 있을 수 있습니다. 투자 결정 및 그에 따른 손익은 전적으로 사용자 본인의 책임입니다.
